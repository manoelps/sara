# [SARA](http://github.com/omadson/sara)

Sistema de Automação Residencial com a utilização da plataforma Arduíno.<br>

## Componentes

* Página de login
* Painel geral
* Página de subsistemas

## Tecnologias
### Linguagens
* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* Python (Somente para criar uma porta serial virtual)

### Frameworks
* JQuery Mobile
